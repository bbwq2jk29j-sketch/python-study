## Day 01 - 파이썬 공부 시작

# 학습 배경
전공 수업과 앞으로의 진로를 고려했을 때 프로그래밍 언어를 기초부터 제대로 익혀야겠다고 판단했다. 특히 데이터 분석과 수학 기반 문제 해결에 관심이 있어서 그 기반 언어로 많이 사용되는 파이썬을 먼저 공부해보기로 결정했다.

# 학습 자료
점프 투 파이썬 교재, 조코딩 유튜브 파이썬 강의

# 오늘 공부한 내용
1. 파이썬이란?
파이썬은 사람이 읽고 쓰기 쉬운 문법을 가진 프로그래밍 언어이다. 복잡한 문법보다 아이디어와 논리 구현에 집중할 수 있다는 점이 인상 깊었다.
2. 파이썬으로 할 수 있는 것들
데이터 분석 및 시각화, 인공지능/머신러닝, 웹 서비스 개발, 자동화 프로그램 제작

오늘 공부한 내용을 보니 특히 데이터 분석 분야에서 수학적 개념을 코드로 바로 표현할 수 있다는 점이 흥미로웠다.

# 내가 느낀 파이썬의 특징
파이썬은 컴퓨터에게 일을 시키는 언어이고, 사람이 읽기 쉬운 문법을 가지고 있다.
그리고 입력-> 처리-> 출력 구조가 명확하다.
아직 처음이라 낯선 것들도 있지만, 설명 자체는 직관적이라 학습에 대한 부담이 크지는 않았다. 단순히 코드를 치는 것보다 개념을 이해하고 기록하는 과정이 중요하다는 것을 느꼈다.
